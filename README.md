# 4-6-8 Breathing Exercise

A calming 4-6-8 breathing exercise to slow your body and settle your mind.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
npm install
npm run dev
```

## Deployment

This project is configured for subpath hosting at `/4_6_8_breathing`.
